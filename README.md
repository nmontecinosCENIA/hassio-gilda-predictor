# 🔌 GILDA Electricity Predictor - Home Assistant Add-on

Este repositorio contiene un add-on personalizado para ejecutar un **modelo de predicción de electricidad** del proyecto **GILDA** dentro de Home Assistant OS.

## 🚀 Instalación rápida

Haz clic en el siguiente botón para agregar este repositorio a tu Home Assistant:

[![Añadir a Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/install_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FnmontecinosCENIA%2Fhassio-gilda-predictor)

Luego, abre **Supervisor > Add-on Store**, busca `predictor_electricity` e instálalo.

## 🗂 Estructura

El add-on se encuentra en:

predictor_electricity/
├── config.yaml
├── Dockerfile
├── predictor_script.py
├── requirements.txt
└── run.sh
