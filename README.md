#  GILDA Electricity Predictor - Home Assistant Add-on

Este repositorio contiene un add-on personalizado para ejecutar  **modelos de predicción** del proyecto **GILDA** dentro de Home Assistant OS.

## Instalación rápida

Haz clic en el siguiente botón para agregar este repositorio a tu Home Assistant:

[![Open your Home Assistant instance and show the add add-on repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FnmontecinosCENIA%2Fhassio-gilda-predictor%2F)


## Estructura

El add-on se encuentra en:

```
predictor_electricity/
├── config.yaml
├── Dockerfile
├── predictor_script.py
├── requirements.txt
└── run.sh
```
